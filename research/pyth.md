# PYTH — Pyth Network

**Current classification**: sector_code=`App Utilities (4)`, chain_ecosystem=`SOL`
**Current sector cohesion**: -0.018, NEG (parent sector)

## What it is

Pyth Network is a low-latency, first-party oracle protocol publishing
price data from ~100+ institutional market-makers and exchanges. Uses a
pull-based "Pythnet" appchain + Wormhole to deliver sub-second feeds to
50+ chains. Dominant oracle on Solana and high-frequency DeFi venues.

## Value accrual mechanism

PYTH governs the protocol and underpins the Oracle Integrity Staking (OIS)
system where stakers back data-publisher accuracy and earn rewards / are
slashed on misreports. The Pyth DAO is moving toward charging data consumers
(institutional subscription model launched 2024-2025); fee revenue flows to
publishers and stakers. No burn currently.

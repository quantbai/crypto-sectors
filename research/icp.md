## What it is

Internet Computer is a non-EVM L1 built by Dfinity Foundation (Dominic
Williams), launched May 2021. Subnet architecture with canister
("smart contract") model that can serve HTTP requests directly. Pitched
as a decentralized cloud / hosting layer (apps run entirely on-chain).
Notable for its catastrophic post-launch price collapse (-99% from ATH).

## Value accrual mechanism

(1) "Cycles" model: ICP burned to mint cycles, which pay for compute /
storage (reverse-gas — devs pay, not users), (2) staking via Network
Nervous System (NNS) with locked maturity, (3) inflation funds node
provider rewards. Value accrual is structurally interesting (burn-on-
compute) but on-chain activity remains low, and the overhang of early
investor unlocks weighed heavily on the chart for years.

## What it is

LayerZero is an omnichain interoperability protocol connecting 160+
blockchains via a generalized cross-chain messaging primitive
(Ultra Light Node + Decentralized Verifier Network model). ZRO is the
native token, launched June 2024.

## Value accrual mechanism

Following community governance Referendum #3 (December 2025), the fee
switch is active: protocol messaging fees, plus all fees from the
adjacent Zero L1 network (priority, MEV, tips), are pooled, converted to
ZRO, and burned. ZRO holders vote every six months on activating /
deactivating the fee switch. This creates a direct usage → burn flywheel
tied to cross-chain message volume.

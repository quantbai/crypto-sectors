## What it is

Aethir is a decentralized GPU cloud / DePIN aggregating idle high-
performance GPUs across 90+ countries for AI inference and cloud gaming.
~435K GPU containers, 150+ enterprise clients, ARR reportedly $147M+.
Source: [Aethir blog](https://ecosystem.aethir.com/blog-posts/aethirs-12-month-strategic-roadmap-supercharging-enterprise-ai-compute-growth).

## Value accrual mechanism

ATH (42B fixed supply, ~12B circulating) is used for: (1) payment for
GPU compute, (2) Checker Node staking (~91K nodes stake ATH to verify
service quality, slashable), (3) Container/Cloud Host rewards. Direct
fee-revenue → token model; "tokenized GPU compute as liquid AI
infrastructure" per Aethir framing. Source: [Aethir blog](https://ecosystem.aethir.com/blog-posts/tokenized-gpu-compute-ath-as-liquid-ai-infrastructure).

## What it is

SENT is the native token of the Session Network — the new chain backing
Session (decentralized encrypted messenger) and successor to the Oxen
project. Migration from OXEN → SENT completed May 2025. EVM-compatible
token standard. Source: [getsession.org](https://getsession.org/migrating-from-the-oxen-network-to-session-network).

## Value accrual mechanism

SENT is staked by Session Node operators (Service Nodes) that route
encrypted messages and onion traffic; nodes earn SENT rewards for
service. Users do not pay per message (messaging is free) — instead,
network usage demand and Service Node deployment economics drive token
demand via collateral lockup. Limited fee-burn; closer to a
"DePIN-for-messaging" emission model.

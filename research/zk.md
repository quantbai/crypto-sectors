## What it is

ZKsync (Matter Labs) is a zkEVM L2 on Ethereum. ZK token launched June
2024 via airdrop; 21 B fixed supply. The protocol is evolving toward
"ZK Stack" / "Elastic Network" — many ZK chains sharing settlement via
the ZKsync Gateway.

## Value accrual mechanism

Originally pure-governance. The "ZKnomics" proposal (live community
debate as of 2025-26) shifts ZK to: (1) required gas token for the
ZKsync Gateway settlement layer, (2) protocol revenue from on-chain
interoperability fees + off-chain enterprise licensing → buyback-and-
burn ZK + fund staking rewards. If passed, this would meaningfully
strengthen accrual.

## Classification verdict

- **Keep current** (202010 Network Scaling). "EVM L2s" sub-group.
- Rationale: ZK trades as an ETH-L2 governance token. Even with ZKnomics,
  it sits squarely with LINEA/STRK/POL. The Elastic Network / interop
  angle is a future cross-chain story but is currently subordinate to L2
  beta.
